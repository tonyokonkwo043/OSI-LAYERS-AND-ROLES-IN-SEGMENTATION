# OSI Layers & Roles in Network Segmentation

## 📌 Overview
This project demonstrates the OSI model layers and their roles in designing and implementing network segmentation. It includes practical Python simulations, diagrams, and notes for understanding how each layer contributes to secure network architecture.

---

## 🛠️ Key Concepts
- OSI Layers (Layer 1–7) and their functions
- Role of each layer in traffic control and segmentation
- VLANs, subnets, and access control at different layers
- How segmentation improves security, performance, and manageability

---

## 📂 Project Files
- `osi_segmentation_demo.py` → Python simulation showing traffic handling per OSI layer
- `diagrams/` → Visual diagrams illustrating OSI layers and segmentation
- `notes/osi_layer_roles.md` → Detailed explanation of each OSI layer and security role

---

## 🎯 Key Learning
- Learned how to map OSI layers to practical network segmentation
- Simulated packet flow and filtering at multiple layers
- Understood how segmentation prevents lateral movement in networks
